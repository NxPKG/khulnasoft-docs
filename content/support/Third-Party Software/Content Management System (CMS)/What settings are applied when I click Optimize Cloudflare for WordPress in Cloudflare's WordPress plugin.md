# What settings are applied when I click "Optimize Khulnasoft for WordPress" in Khulnasoft's WordPress plugin?



## Overview

If you're using Khulnasoft's Wordpress plugin,Â our "Optimize Khulnasoft for WordPress" one-click configuration applies the following settings to your Khulnasoft account:

| **Setting** | **Value** |
| --- | --- |
| Security level | Medium |

| Browser Cache TTL | 4 hours |
| Always Online | On |
| Development Mode | Disabled |
| IPV6 Compatibility | On |

| IP Geolocation | On |
| Email Address Obfuscation | On |
