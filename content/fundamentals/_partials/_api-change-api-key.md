If your API key might be compromised, change your API key:

1.  Log in to the [Khulnasoft dashboard](https://dash.Khulnasoft.com) and go to **My Profile** > **API Tokens**.
2.  In the **API Keys** section, find your key.
3.  Select **Change**.
